# Complexity allowlist for `scripts/check-complexity.sh`.
#
# Format:  <crate>|<max-allowed-violations>|<reason>
#
# `max-allowed-violations` is the per-crate ceiling for
# `clippy::too_many_lines` warnings (default threshold 100 LOC per
# function). When a crate's count exceeds its ceiling, the gate
# fails — the diff that introduced the regression is required to
# either refactor the new oversized function OR raise this ceiling
# with a justification in the reason field.
#
# Refactoring a function below 100 LOC drops the count → please
# also drop the ceiling here so the new floor is locked in.
#
# The initial baseline was captured 2026-04-27 (post-SOTA
# Tier 1 + Tier 2 delivery). Every entry pairs to a per-crate
# follow-up to refactor the offending functions during the next
# decomposition sprint (sunset alongside the size-allowlist
# entries at 2026-07-23).
