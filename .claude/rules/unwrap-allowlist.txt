#
# Rules:
#   - Every entry MUST carry a concrete reason that explains why an unwrap
#     is safer than a typed error (typically a type-system-inexpressible
#     invariant that is proven elsewhere in the file).
#   - Every entry MUST carry a sunset date. The gate fails when today > sunset.
#   - Whole-file allowlisting is supported (omit `:line`) for test-fixture
#     crates where every site is a `Mutex::lock().unwrap()` and rewriting
#     to `?` would add noise without safety gain.
#   - Prefer refactoring over allowlisting for production code.
