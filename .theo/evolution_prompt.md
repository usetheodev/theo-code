# Evolution Prompt

Evolua o context manager do theo-code. O sistema de context management inclui:
- `GraphContextProvider` trait (theo-domain)
- `GraphContextService` impl (theo-application)
- `ContextAssembler` (theo-application)
- `ContextMetrics` (theo-agent-runtime)
- `WorkingSet` (theo-domain)

Pesquise padrões SOTA nos repos de referência (especialmente OpenDev, QMD, Pi-Mono) e aplique melhorias que tornem o context manager mais robusto, eficiente e alinhado com o estado da arte.
